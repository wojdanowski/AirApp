
GET localhost:8000/ - ping serwera
GET localhost:8000/api - ping api
GET localhost:8000/api/stations - lista dostępnych stacji
GET localhost:8000/api/nearestAirIndex?lat=X&lon=Y - dane o powietrzu ze stacji najbliższej do XY

Te będą jeszcze trochę zmodyfikowane (np. nie po id):
POST localhost:8000/subscriptions/ - dodaje nową subskrypcję
GET localhost:8000/subscriptions/<id> - pobiera subskrypcję dla id
PATCH localhost:8000/subscriptions/<id> - zmienia subskrypcję
PUT localhost:8000/subscriptions/<id> - zmienia subskrypcję
DELETE localhost:8000/subscriptions/<id> - usuwa subskrypcję subskrypcję

Prawidłowy json do zapisania subksrypcji:
{
    email: "test@wp.pl",
    lat: 20.1234,
    lon: 51.1234,
    hours: [10, 12]
}

Prawidłowy json do edycji subksrypcji:
{
    lat: 20.1234,
    lon: 51.1234,
    hours: [10, 12]
}
